Getting Started

  • Enable location permissions for real-time friend tracking and map accuracy
  • Turn on “Share my location” to appear on the live festival map
  • Add friends using your unique friend code
  • Star artists to build your personal schedule

Common Issues

  • Location not updating: Ensure “Always Allow” location access is enabled
  • Friends can’t see me: Confirm location sharing is enabled and friendship is accepted
  • Missing notifications: Check system notification permissions for Festi
  • Map not loading: Restart the app or verify network connection

Account & Data

You can fully manage your data inside the app:

Settings → Account & Data → Delete account

Deleting your account permanently removes all profile data, friends, location history, and activity stats.
